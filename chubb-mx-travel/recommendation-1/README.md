# Recommendation 1

## Cliente
CHUBB MX Travel

## Hipótesis
Si organizamos las coberturas en categorías con una jerarquía visual clara, los usuarios podrán localizar más rápidamente la información relevante y comprender el valor del plan con menor esfuerzo, incrementando la selección de planes.

## Objetivo
Objetivo: Reducir la carga cognitiva durante la selección de planes y facilitar la comparación entre opciones para aumentar la selección de planes y disminuir el abandono en este paso del funnel.

## Cambios
Las coberturas se agrupan en 3 grupos: “Atención médica”, “Viaje y equipaje” y “Protección adicional”.

## Tecnologías
- HTML
- JavaScript (incluye los estilos CSS)
