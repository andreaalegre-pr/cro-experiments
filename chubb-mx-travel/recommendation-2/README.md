# Recomendación 2:
Home: Optimización del ATF (Above the fold)

## Cliente
CHUBB MX

## Hipótesis
Si reducimos la altura del banner principal y adelantamos visualmente el formulario (cotizador) para que quede visible desde el primer scroll, entonces aumentará la interacción con el formulario y el avance hacia la página de planes.

## Objetivo
Incrementar la cantidad de usuarios que interactúan con el cotizador y avanzan hacia la página de planes, reduciendo el rebote y mejorando la visibilidad de los elementos clave de conversión.

## Tecnologías
- JavaScript (incluye los estilos CSS)
